# Simple Express API

A basic Node.js + Express API project.

## Features
- `GET /` → Welcome message
- `GET /api/hello` → Returns JSON: `{ "message": "Hello, World!" }`
- `POST /api/data` → Accepts JSON `{ "name": "YourName" }` and responds with a greeting

## Setup & Run
```bash
npm install
node server.js
